# MRI_y_BAEPS
Descripción de los datos:
BAEPS audio*: datos de la fase audiológica electrofisiología de potenciales evocados auditivos de jóvenes y adultos mayores.
BAEPS neuro*: datos de la fase neurológica electrofisiología de potenciales evocados auditivos de jóvenes y adultos mayores.
MRI*: datos de imagen por resonancia magnética de jovenes y adultos mayores.
